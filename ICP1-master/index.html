How to create a GitHub Repositorie

Download GitHub Desktop

  • follow the promp and install the desktop apt
  • once the apt is open click "file" --> "New Repository"
  • Here is my repository for this project

Steps to insure a working site.

  1. Make sure your index.html file is in the root site
    since my root site was the repository M1-ICP1 my site would not
    load as long as my index.html was in source directory
  2. Use the Copy path
    Ensure your links are correct by using the copy path function
    on the github page
  3. Setup github to host
    Navigate to the settings tab on gethub and make sure it is
    set up to host your repository via their sever
